🍽️ Restaurant Order API

This is a simple REST API built using ASP.NET Core Web API.

## Features :
- Create Order (POST)
- Get All Orders (GET)
- Get Order by Id (GET)
- Update Order (PUT)
- Delete Order (DELETE)

## Technologies Used :
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger (for API testing)

## How to Run :
1. Clone the repository
2. Open project in Visual Studio
3. Update database connection string
4. Run migrations:
   - Add-Migration InitialCreate
   - Update-Database
5. Run the project

## API Testing :
You can test all APIs using Swagger UI:
https://localhost:7012/swagger/index.html
